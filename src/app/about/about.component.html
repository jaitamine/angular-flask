machine learning project

by health programming team

this project is a remote access platform dedicated to detection , classification and visualization of cervical hemorrhages. It is made up of three tools:
  • Safe storage space for data and their organization.
  • Tool for viewing existing scanners in the storage space.
  • Help tool for the detection of haemorrhage as well as the classification of their types ((epidural, intraparenchymal, intraventricular, subarachnoid, subdural).
